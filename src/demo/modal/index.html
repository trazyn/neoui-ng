Example



AngularJS
Service * $ui.modal
angular .module( "app", [ "$ui.modal" ] ) .controller( "modalController", [ "$scope", "$modal", function( $scope, $modal ) { $scope.name = "trazyn"; $scope.open = function( animate ) { $modal.open( { controller: "modalController", template: "{{ name }}<br/><p>This is a modal window. You can do the following things with it:</p><ul> <li><strong>Read:</strong> modal windows will probably tell you something important so don't forget to read what they say.</li> <li><strong>Look:</strong> a modal window enjoys a certain kind of attention; just look at it and appreciate its presence.</li> <li><strong>Close:</strong> click the outside close the modal.</li> </ul>", animate: animate } ); }; $scope.showProgress = function() { $modal.open( { controller: "modalController", showProgress: false, templateUrl: "src/demo/modal/page.html", title: "Instagram Handbook for Brands", class4modal: "demo", scope: $scope } ) .progress.start(); }; } ] );
Methods

$modal.open( options ) - 打开弹出框

options - Object

scope - null

指定scope


其他见Options


jQuery
Constructor

$.modal( options )

options - 参数设置

$.modal( { animate: "slide" } );
Options

title - "Modal.JS"

指定title


render - null

弹出框内容, 字符串或function


class4modal - ""

指定弹出框样式


selector4drag - false

拖拽元素选择器, 为true时, 通过标题栏拖拽


css - {}

添加行内样式


attr - {}

添加属性


autoShow - true

初始化后打开弹出框, 不通过实例open方法


showHead - true

显示标题头


showOverlay - true

显示遮罩层


showProgress - true

Show the progress bar


fadeIn - true

内容淡入


closeByESC - true

Press the Esc close the modal


closeByDocument - true

点击弹出框外面关闭弹出框


animate - "slide"

弹出框样式, slide, stick, bottom, flip, vertical, sign, scale, fadein


unload - null

弹出框卸载触发
