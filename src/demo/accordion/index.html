Javascript
angular .module( "demo.accordion", [ "$ui.accordion" ] ) .controller( "accordionController", [ "$scope", function( $scope ) { $scope.multiple = true; $scope.onExpand = function( index ) { console.log( "Expand: " + index ); }; $scope.onCollapse = function( index ) { console.log( "Collapse: " + index ); }; $scope.panes = [ { head: "Pane 1", content: "Pane - 1" }, { head: "Pane 2", content: "Pane - 2" } ]; $scope.addPane = function() { var index = +new Date(); $scope.panes.push( { head: "Pane " + index, content: "Pane - " + index } ); }; $scope.removePane = function() { $scope.panes.splice( -1 ); }; } ] );
Directive * s-accordion

=multiple - false

同时展开多个内容区域


=controller - null

Controller of the accordion, See Method


&on-expand - null

展开时回调


&on-collapse - null

收起时回调


Directive * s-accordion-pane

@is-open - false

是否展开


@head - null

标题, 不为空时覆盖s-accordion-head


@content - null

内容, 不为空时覆盖s-accordion-content, 默认取值s-accordion-pane中raw值


@index - null

索引, 可选


Directive * s-accordion-head

自定义标题栏


Directive * s-accordion-content

自定义内容

Ut molestie a, ultricies porta urna. Quisque lorem tortor fringilla sed, vestibulum id. </div> </div> </div>


Constructor

$( selector ).accordion( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.accordion" ).accordion();


Options

multiple - false

同时展开多个内容区域


duration - 300

动画速度, 单位毫秒


onExpand - null

展开时回调


onCollapse - null

收起时回调


selector4pane - ">.pane"

Selector of pane


selector4head - ".head:first"

Selector of the head


selector4content - ".content:first"

Selector of the content




Method

accordion.expand( index )

index - Number

展开指定区域


accordion.collapse( index )

index - Number

收起指定区域


accordion.toggle( index )

index - Number

展开/收起指定区域


accordion.expandAll() - 展开所有


accordion.collapseAll() - 收起所有
