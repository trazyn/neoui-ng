Example


API
HTML
<div class="ui tab" style="min-height: 300px;"> <div class="nav"> <div class="item selected" data-index="github">Github</div> <div class="item selected" data-index="dribbble">Dribbble</div> <div class="item selected" data-index="twitter">Twitter</div> </div> <div class="content"> <section data-index="github"></section> <section data-index="dribbble"></section> <section data-index="twitter"></section> </div> </div>

Ajax页面的加载, 添加data-page="url"属性

Constructor

$( selector ).tab( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.tab" ).tab( { duration: 500 } );
Options

rule - "data-index"

页签和内容映射


class4loading - "loading"

Ajax加载内容时作用于页签


class4error - "error"

Ajax加载失败时作用于页签


ripple - true

页签点击大波浪


duration - 300

动画时间, 单位毫秒


onSelect - null

取值为function时, 页签选择时执行该方法, 取值为Hash对象时, 通过data-index执行对应回调

$( ".ui.tab" ).tab( { onSelect: { "github": function( tab, settings ) { console.log( "Github" ); }, "dribbble": function( tab, settings ) { console.log( "Dribbble" ); } } } );
Method

tab.add( items )

items - Object/Array

新增一个或多个tab

{ name - 新增页签显示, 取值不限, 可以为 HTML index - 页签索引值 render - 内容渲染, 返回内容 immediate - 是否立即加载内容, 默认 false }

tab.remove( index )

index - String

移除指定tab


tab.getTab( index )

index - String

获取指定tab的DOM


tab.active( index )

index - String

选中对应tab


tab.isActive( index )

index - String

判断对应tab是否被选中


tab.disabled( [index] )

index - String/Array

禁用tab, 参数空时禁用所有tab


tab.enabled( [index] )

index - String/Array

取消禁用, 参数空时取消所有已禁用tab
