Example


AngularJS
Markup
<ng-tab-set tab-selected="selected" tab-on-selected="onSelected"> <ng-tab ng-repeat="tab in tabs" tab-index="tab.index" tab-header="\{\{ tab.title \}\}"> <div ng-bind-html="\{\{ tab.content \}\}"></div> </ng-tab> </ng-tab-set>
Javascript
angular .module( "app", [ "$ui.tab" ] ) .controller( "tabController", [ "$scope", function( $scope ) { /** Set the default selected */ $scope.selected = "1"; $scope.tabs = [ { header: "Tab 1", content: "Content 1", index: "1", disabled: true }, { header: "Tab 2", content: "Content 2", index: "2" } ]; $scope.onSelected = function( scope ) { console.log( "Current selected: " + scope.index ); }; } ] );
Directive * ng-tab-set

=tab-selected - "0"

默认选中项


@tab-no-ripple - null

If present, disable ink ripple effects


@tab-no-lavalamp - null

设置true后取消滑动效果


&tab-on-selected - null

选中后执行回调


@tab-vertical - false

设置true后垂直显示

Directive * ng-tab

@tab-index - null

页签索引, 不可为空或重复


=tab-disabled - false

禁用页签


@tab-header - null

页签标题


@tab-template-url - null

通过指定URL加载内容



jQuery
HTML
<div class="ui tab" style="min-height: 300px;"> <div class="nav"> <div class="item selected" data-index="github">Github</div> <div class="item selected" data-index="dribbble">Dribbble</div> <div class="item selected" data-index="twitter">Twitter</div> </div> <div class="content"> <div data-index="github"></div> <div data-index="dribbble"></div> <div data-index="twitter"></div> </div> </div>

Ajax页面的加载, 添加data-page="url"属性

Constructor

$( selector ).tab( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.tab" ).tab( { duration: 500 } );
Options

rule - "data-index"

页签和内容映射


class4loading - "loading"

Ajax加载内容时作用于页签样式


class4error - "error"

Ajax加载失败时作用于页签样式


ripple - true

If present, enable ink ripple effects


duration - 300

设置lavalamp后, 动画时间, 单位毫秒


onSelect - null

取值为function时, 页签选中后执行该方法, 取值为Hash对象时, 通过data-index执行对应回调

$( ".ui.tab" ).tab( { onSelect: { "github": function( tab, settings ) { console.log( "Github" ); }, "dribbble": function( tab, settings ) { console.log( "Dribbble" ); } } } );
Method

tab.add( items )

items - Object/Array

新增一个或多个tab

{ name - 新增页签显示, 取值不限, 可以为 HTML index - 页签索引值 render - 内容渲染, 返回内容 immediate - render不为空时, 是否立即加载内容, 默认 false page - 通过指定URL加载内容 }

tab.remove( index )

index - String

移除指定tab


tab.getTab( index )

index - String

获取指定tab的DOM


tab.active( index )

index - String

选中对应tab


tab.isActive( index )

index - String

判断对应tab是否被选中


tab.disabled( [index] )

index - String/Array

禁用tab, 参数空时禁用所有tab


tab.enabled( [index] )

index - String/Array

取消禁用, 参数空时取消所有已禁用tab
