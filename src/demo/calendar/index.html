AngularJS
Markup
<s-calendar ng-disabled="isDisabled" ng-model="date" double="double" placeholder="Year-Month-Day" min-date="minDate" max-date="maxDate" on-selected="onSelected" default-date="defaultDate"> </s-calendar>
Javascript
angular .module( "demo.calendar", [ "$ui.calendar" ] ) .controller( "calendarController", [ "$scope", function( $scope ) { $scope.init = function() { $.anchor( { offset: -60 } ); }; var now = new Date(); angular.extend( $scope, { date: $.dateutil( now ).tomorrow(), isDisabled: false, minDate: $.dateutil( now ).lastWeek(), maxDate: $.dateutil( now ).nextWeek(), onSelected: function( value ) { console.log( value ); } } ); } ] );
Directive * s-calendar

=ng-disabled - false

禁用控件


=ng-model - null

当前选择时间


&on-selected - null

选中时执行


@format - "%Y-%m-%d"

日期格式化, See Dateutil


=double - false

显示双月


@placeholder - "Year - Month - Day"

The same to input placeholder


=min-date - null

最小可选日期


=max-data - null

最大可选日期


API
HTML
<div class="md-calendar"> <input class="md-text" type="text" placeholder="Year - Month - Day" readonly /> <i class="md-icon-calendar"></i> </div>


Constructor
$( selector ).calendar( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".md-calendar" ).calendar();


Options

daysOfTheWeek - [ "S", "M", "T", "W", "T", "F", "S" ]

日期名称


format - "%Y-%m-%d"

日期格式化, See Dateutil


selectable - null

返回true/false日期是否可选


selectableDOW - [1, 2, 3, 4, 5]

每周中可选星期


double - false

双月显示


defaultDate - new Date()

默认日期


minDate - null

最小可选日期


maxDate - null

最大可选日期


onSelected - nul

$( ".md-calendar" ).calendar( { onSelected: function( value ) { console.log( "Current: " + value ); } } );

选中时执行


showTime - false

显示时间输入框


selector4input - ":input:first"

输入框选择器


selector4trigger - ".md-icon-calendar"

事件触发节点选择器



Method
.val( [ value ] ) - 设置/取值

value - String/Date


.disabled() - 禁用控件

.enabled() - 取消禁用
