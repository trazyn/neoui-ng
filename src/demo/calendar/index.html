Example


API
HTML
<div class="ui calendar"> <input class="ui text" type="text" placeholder="Year - Month - Day" readonly /> <i class="icon calendar"></i> </div>


Constructor

$( selector ).calendar( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.calendar" ).calendar();


Options

months - [ "January", "February", "March", "April", "May", "July", "August", "September", "October", "November", "December" ]

月份显示名称


format - "%Y-%m-%d"

日期格式化, 见 Date Util


selector4input - ".input:first"

输入框


showTime - false

是否显示时间选择栏(注意时分秒的显示是由format参数控制的)


selector4trigger - ".icon.calendar"

事件源



Method

calendar.val( [ value ] )

value - String/Date

设置/取值


calendar.disabled()

禁用控件, .ui.calendar[disabled]等效


calendar.enabled()

取消禁用
