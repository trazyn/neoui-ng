Success

This is a message telling you that everything is a-okay!

AngularJS
angular .module( "app", [ "$ui.message" ] ) .controller( "messageController", [ "$message", function( $message ) { $message.success( "This is a message telling you that everything is a-okay" ); } ] );
jQuery
require( [ "ui/message/message" ], function() { $.message.success( "This is a message telling you that everything is a-okay" ); } );
API

success( content, [ delay ] );

content - String, 文本内容

delay - Number, 自动关闭延时, 单位毫秒

Confirm
Example

AngularJS
angular .module( "app", [ "$ui.message" ] ) .controller( "messageController", [ "$message", function( $message ) { $message.confirm( { title: "Please confirm", message: "Exported successfully. Do you want to open the export query page?", onOk: function() { window.open( "//www.google.com", "_blank" ); } } ); } ] );
jQuery
angular .module( "app", [ "$ui.message" ] ) .controller( "messageController", [ "$message", function( $message ) { $.message.confirm( { title: "Please confirm", message: "Exported successfully. Do you want to open the export query page?", onOk: function() { window.open( "//www.google.com", "_blank" ); } } ); } ] );
API

confirm( options );

title - String, 对话框标题

message - String, 对话框内容

onOk - Function, 点击 执行回调

onClose - Function, 点击 执行回调
