Constructor

$( selector ).validation( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.form.validation" ).validation( { custom : { uniqueName: function( value ) { return value !== "abc"; } }, breakOnError: true } );
Options

selector - ":input[validators]:visible:not(button)"

表单元素选择器


validators -

检验规则, 详见Validators


breakOnError - true

同组校验中, 碰到错误停止


custom - null

自定义校验器(function类型, 可返回$.Deferred对象)映射集


class4error - "error"

校验失败样式


parseElement -

自定义控制元素, 组模式校验中, 返回外层元素, 见源码部分

validation.validate()

validation.clean()

Validators
Regular

required - 必填字段


equals - 等于参数值


contains - 包含指定内容


startWith - 字符串以指定内容开始


endWith - 字符串以指定内容结束


min - 不小于指定值


max - 大于指定值


minLength - 长度不小于指定值


maxLength - 长度小于指定值


past - 日期大于当前


future - 日期小于当前


after - 日期大于指定时间


before - 日期小于指定时间


int - 输入内容为数字


float - 输入内容为浮点


alpha - 输入内容为字母


numeric - 输入内容为数字


hexadecimal - 输入内容为16进制


uppercase - 输入内容大写字母


lowercase - 输入内容小写字母


email - 输入内容为 email


url - 输入内容为 url


zipCode - 输入内容为 zipCode


ip - 输入内容为 ip


phone - 输入内容为手机号码


timeString - 输入内容为时间字符串


dateString - 输入内容为日期字符串
